# Google Search Console Kurulum

1. https://search.google.com/search-console adresine gidin
2. Mülk ekle: `https://adanabosanmaavukati.org`
3. DNS veya HTML doğrulama yapın
4. Site Haritaları → `https://adanabosanmaavukati.org/sitemap_index.xml` gönderin
5. Sayfa dizine ekleme ve FAQ rich result raporlarını haftalık izleyin
